# 🏪 7-Delivery Demo Filter System (Prototype)



โปรเจกต์นี้เป็น **ต้นแบบ (Prototype)** ของระบบฟิลเตอร์และหน้าการชำระเงินจำลอง  
ที่พัฒนาโดยใช้แนวคิด **Design Thinking** เพื่อยกระดับประสบการณ์ผู้ใช้ของแอป **7-Delivery**  
ให้สามารถ “ค้นหาและเลือกสินค้าได้ตรงใจมากขึ้น” ผ่านระบบคัดกรองที่เข้าใจง่ายและสมจริง

---

## 🚀 ฟีเจอร์หลัก (Key Features)

- 🔍 **ระบบฟิลเตอร์สินค้า** — กรองตามหมวดหมู่ โปรตีน ฐานอาหาร ราคา และคำค้นหา  
- 🍱 **หน้าสินค้าอัจฉริยะ (Smart PDP)** — เลือกซอส, อุ่น, เพิ่มไข่ (+5 บาท) ได้จริง  
- 🛒 **ระบบตะกร้าสินค้า (Cart System)** — เพิ่ม-ลบสินค้า, คำนวณราคาอัตโนมัติ  
- 💳 **หน้าชำระเงินจำลอง (Checkout Page)** — กรอกชื่อ-ที่อยู่-เบอร์โทร, เลือกชำระแบบ COD หรือ Coming Soon อื่น ๆ  
- ✈️ **หน้าจัดส่งจำลอง (Delivery Page)** — มีโลโก้เครื่องบินและข้อความขำ ๆ “กำลังไปบินไปส่งของ”  
- 💾 **ระบบบันทึกข้อมูล (LocalStorage)** — จำข้อมูลตะกร้าและที่อยู่ได้แม้รีเฟรชหน้า

## 💡 แนวคิดการออกแบบ (Design Thinking Summary)

1. **Empathize** – สำรวจปัญหาผู้ใช้จริงที่ค้นหาสินค้ายากในแอป 7-Delivery ปัจจุบัน  
2. **Define** – “เราจะทำอย่างไรให้ผู้ใช้ค้นหาอาหารที่ต้องการได้ง่ายและตรงใจขึ้น?”  
3. **Ideate** – ระดมความคิด สร้างระบบฟิลเตอร์ที่ใช้งานง่ายและเป็นมิตร  
4. **Prototype** – สร้างหน้าเว็บจำลองพร้อมระบบกรองสินค้า  
5. **Test** – ทดลองใช้งานจริงและปรับปรุง UX ให้สมจริงที่สุด  

---

## ⚙️ วิธีรันโปรเจกต์ (How to Run)

1. ดาวน์โหลดโฟลเดอร์โปรเจกต์ทั้งหมด  
2. เปิด `index.html` ด้วย **Live Server** บน VS Code  
3. ทดลองค้นหา / กรองสินค้า / เพิ่มลงตะกร้า  
4. ไปที่หน้า **ชำระเงิน (Checkout)** → กรอกข้อมูล → กดยืนยันคำสั่งซื้อ  
5. ระบบจะพาไปหน้า “กำลังจัดส่ง” (Delivery Page)
